Question:

Please do regression by two different algorithms with one dataset. Select one dataset public which consists of more than 5 features. Please do some tasks Exploratory Data Analysis (EDA), feature engineering and some pre-processing if you feel need it. Please explain each your task results

Answer :

What we did is, we tried to create prediction of gold price using regression technique. The datasets coming from public dataset of gold price in rupiah that we downloaded from kaggle (link bellow). First, we did some eda that result in feature selection, data transformation, and data splitting. Second, we used 2 different regression algorith to make prediction model. Finally, we compare prediction result with dataset and evaluate it using mse and r2.

From Explonatory Data Analysis, we could assume that x,y - where x is date and y is data price- has linier corelation. Therefore, we choose linier regression method. For the second algorithm, we choose polynomial regression as it's based on linier regression but support curve line.

The result are polynomial regression has lower mse score and higher r2 score compare to linier regression.

For deeper explaination, please see bellow

# Feature Selection 
df = dt[['Date','United States(USD)']]

# Feature Renaming
df = df.rename(columns= {
    'Date': 'x',
    'United States(USD)': 'y',
})

# Change date to numerical format
# dd-mm-yyyy => yyyymm
for i in range(len(df)):
    a = df.loc[i,'x'] 
    a = a.split('-')
    a = int(a[2] + a[1])
    df.loc[i, "x"] = a

In data processing we use 2 different algorithm, namely liner regression and polynomial regression.

from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline
from sklearn.metrics import mean_squared_error, r2_score

import matplotlib.pyplot as plt

# Setup degree
degree = 2
regressor_pl = make_pipeline(PolynomialFeatures(degree), LinearRegression())
regressor_pl.fit(x_train,y_train)

# Predict linier
Y_pred_r_train = regressor_pl.predict(x_train)
y_pred_r = regressor_pl.predict(x_test)

print("Linier Regression Results")
# print(" Coefficient : ", regressor_linier.coef_)
print(" MSE         : ", mean_squared_error(y_test, y_pred_r))
print(" R2          : ", r2_score(y_test, y_pred_r))


# Draw Graph
plt.scatter(x_test, y_test, color='black', label="actual")
plt.plot(x_test, y_pred_r, linewidth=1, label="predicted")
plt.title("Polynomial Regression")
plt.legend()
plt.show()
